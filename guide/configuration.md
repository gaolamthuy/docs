# Cấu hình

Hướng dẫn cấu hình VitePress và DecapCMS.

## Cấu hình VitePress

File cấu hình chính: `.vitepress/config.js`

## Cấu hình DecapCMS

File cấu hình: `admin/config.yml`

## Assets

Tất cả assets (hình ảnh, logo, icons) được lưu trong thư mục `assets/`.

### Sử dụng assets trong markdown

![Logo](/assets/logos/logo-main-hexagon-extrawhiteborder-forproductimage-resize-200x200.png)
